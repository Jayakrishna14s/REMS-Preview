About

ESTATEHUB

The Estatehub (Real Estate Management System) streamlines the property buying and selling process, creating a seamless experience for both sellers and buyers. Sellers can easily upload their property details, including images, descriptions, and pricing, through a user-friendly interface. Buyers can browse these listings using various filters such as location, price range, and property type to find their ideal home or investment property. Once a property is selected, the platform facilitates secure and straightforward transactions, making the entire process efficient and hassle-free. This system bridges the gap between sellers and buyers, offering a one-stop solution for real estate needs.

Development

JSP

JSP was used to create dynamic and interactive web pages for the system. It enabled us to design a user-friendly interface for seller to upload their property details and for buyers to browse and choose properties.

Servlets

Servlets were employed to handle the core business logic and process requests from users. They act as the backbone of the application, managing interactions between the user interface and the database.

Tomcat

Apache Tomcat served as the application server to deploy and run the web application. It provided a reliable and scalable environment to handle user requests efficiently.

MySQL

MySQL was used as the database to store and manage all the property listings, user details, and transaction records securely.

Deployment

Docker

Docker was used to containerize the application, encapsulating all its dependencies and configurations into a portable image. This allowed for easy deployment on the Render platform, ensuring a consistent and efficient runtime environment.

Docker Compose

Docker Compose was used to manage multiple services of the application, such as the tomcat server and database. By defining these services in a docker-compose.yaml file, the system could be deployed with a single command, enabling seamless orchestration and scalability.
